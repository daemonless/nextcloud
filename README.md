## Deployment

### Podman Compose

```yaml
services:
  nextcloud:
    image: ghcr.io/daemonless/nextcloud:latest
    container_name: nextcloud
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - "/path/to/containers/nextcloud:/config"
      - "/path/to/containers/nextcloud/data:/data"
    ports:
      - 8082:8082
    restart: unless-stopped
```
